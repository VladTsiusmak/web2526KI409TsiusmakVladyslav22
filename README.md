Hardware -lab1 :

Для створення вбудованого вебсервера використовується таке апаратне забезпечення:
Мікроконтролер ESP32 (версія ESP32 DevKit V1) — виконує функції автономного HTTP-сервера та обробляє запити від клієнтів.
Інтегрований Wi-Fi модуль — забезпечує підключення пристрою до бездротової мережі та отримання мережевої IP-адреси.
Flash-пам’ять — застосовується для зберігання файлів вебінтерфейсу та інших необхідних даних.
USB-інтерфейс — використовується для подачі живлення на плату, а також для завантаження прошивки через вбудований перетворювач.

Програмні засоби та бібліотеки (lab1):
HTML та CSS — застосовуються для розробки структури та оформлення вебсторінки керування.
Arduino Framework — середовище, що використовується для написання та компіляції програмного коду для ESP32.
LittleFS — файлова система, оптимізована для мікроконтролерів, яка дозволяє зберігати статичні вебфайли у Flash-пам’яті пристрою.
WiFi.h — бібліотека, що забезпечує роботу мережевого підключення, дозволяє налаштовувати Wi-Fi модуль у режимі клієнта та під’єднуватися до точок доступу.
WebServer.h — використовується для розгортання HTTP-сервера на порту 80 та обробки HTTP-запитів (наприклад GET), які надсилає веббраузер.
